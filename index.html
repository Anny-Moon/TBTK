Welcome to the project page for TBTK

TBTK is a c++ library for solving general bilinear Hamiltonians, and a flexible indexing scheme allows for general models to be setup with little effort. The model can either be solved using a DiagonaliztionSolver, giving eigenstates and eigenvectors, or using a ChebyshevSolver to calculate Green's functions.
