Welcome to the Tight Binding Toolkit - TBTK

For an introduction to the library, see http://dafer45.github.io/TBTK

###########################
# Installation instructions
###########################
Without CUDA (gpu)  
source init_session.sh  
./install.sh

With CUDA (gpu)  
source init_session.sh  
./install -CUDA

#############
# Update TBTK
#############
./update.sh

####################
# Initialize session
####################
source init_session.sh
